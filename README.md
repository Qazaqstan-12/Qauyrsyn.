# Qauyrsyn.
«Qauyrsyn» — қанатты достар әлеміне жол көрсетушіңіз. Біздің платформа — құстардың құпиясы мен сұлулығын ашатын таптырмас кеңістік. Қазақстанның және әлемнің түкпір-түкпіріндегі құстар туралы қызықты деректер, орнитологиялық зерттеулер мен таңғажайып фото-видео материалдар енді бір жерде жинақталған. Өлкеңіздің табиғатын бізбен бірге таныңыз!
